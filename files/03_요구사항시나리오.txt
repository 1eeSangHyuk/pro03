요구사항 시나리오

@회원가입
회원 가입시 중복 체크는 json를 활용하여 별도의 팝업 창이 없이 현재 창에서 구현
회원 가입시 주소의 입력은 daum 주소 api를 활용하여 주소를 전달하여 구현
회원의 비밀번호는 aes256 등으로 암호화하여 처리하여 구현
--회원 가입 후에는 네이버 메일서버로 가입 축하 이메일 발송되도록 구현
@@관리자
관리자는 회원의 직권 등록, 직권 탈퇴 및 정보 수정이 가능하도록 구현

@공지사항
공지사항은 제목, 내용 등과 같이 파일 첨부(업로드) 기능을 함께 구현
공지사항 전체 목록의 페이징 처리와 검색 기능 구현
공지사항 글의 상세보기 화면에서 해당 게시된 공지 중에서 파일 다운로드 기능 추가
질문 및 답변 기능은 회원은 해당 질문시 제목과 내용을 작성하고, 관리자는 그 질문에 대한 답변을 작성할 수 있도록 구현

@Q&A
질문 및 답변의 목록은 해당 질문별로 그룹화 되어 답변이 해당 질문의 밑에 오도록 구현
@@관리자
관리자는 답변, 질문 삭제가 가능하도록 구현

@여행 상품
여행 상품 정보에는 상품 코드, 카테고리코드, 상품명, 여행기간, 상품 설명, 참가인원 등으로 구성되어 등록되도록 구현
여행상품 정보는 기본적인 목록으로 구성하되 메뉴에서 해당 카테고리를 선택하면, 그에 맞는 카테고리의 상품을 목록으로 제공
여행상품 상세보기 화면에서 해당 여행후기를 표시하고, 여행상품 상세보기에서 신청 또는 좋아요 기능 구현
@@관리자
관리자는 여행의 목록, 검색, 등록, 정보 수정, 삭제 등이 가능하도록 구현
관리자는 여행 신청 정보에 대한 등록, 수정, 철회 등이 가능하도록 구현

@후기
이용후기는 댓글과 별점 등으로 구성되어 구입한 상품에 대하여만 등록될 수 있도록 구현
이용후기의 수정은 해당 여행을 신청하고 여행한 회원만 수정할 수 있도록 구현
모든 목록은 페이징 처리가 될 수 있도록 구현
@@관리자
관리자는 후기 삭제가 가능하도록 구현
